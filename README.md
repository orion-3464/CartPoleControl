# Installation
1. Install [Webots](https://cyberbotics.com/).
   - **Windows:** Download installer from the link above.
   - **MacOS:** Download Bundle file from the link above.
   - **Linux:** Follow the [Instructions](https://cyberbotics.com/doc/guide/installation-procedure#installation-on-linux)
2. Launch Webots. Then ```File->Open World``` and choose the world file (```InvertedPendulumWorld.wbt```)
